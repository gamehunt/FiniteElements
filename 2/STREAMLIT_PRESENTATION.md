# Streamlit-доклад по задаче обтекания двух цилиндров

## Запуск

Из корня репозитория:

```bash
streamlit run 2/presentation/app.py
```

## Содержание доклада

Доклад оформлен как учебное представление численного решения задачи в области с двумя цилиндрами.

Структура:

- титульный слайд;
- постановка задачи;
- расчётная область;
- граничные условия и параметры задачи;
- расчётные сетки;
- конечно-элементная аппроксимация;
- выбор степени полинома `p = 1, 2, 3`;
- численные решения для симметричных и несимметричных сценариев;
- сравнение результатов и интегральных характеристик по границам цилиндров;
- краткие выводы.

## Расчёт во время работы приложения

Основной режим работы — runtime-расчёт.

- для базовых и симметричных сценариев вызывается `solve_problem(...)` из `2/solve.py`;
- для несимметричных сценариев вызывается `solve_problem_non_symmetrical(...)` из `2/solve_non_symmetrical.py`;
- степень аппроксимации `p` выбирается при запуске расчётных разделов и напрямую передаётся в вычислительную ветвь;
- в разделе результатов показываются поле решения и интегральные характеристики по границам цилиндров;
- для несимметричных сценариев дополнительно выводятся коэффициенты компенсации `kappa_1`, `kappa_2`.
