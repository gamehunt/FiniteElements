# Streamlit-презентация по задаче из проекта

## Что добавлено

В каталог `2/` добавлена отдельная Streamlit-презентация:

- `presentation/app.py` — основной entry point;
- `presentation/presentation_data.py` — чтение параметров `.geo`, чтение сеток `.xml`, построение схем и сеточных иллюстраций.

Презентация не меняет математическую модель и не вмешивается в существующий `solve.py`.
Она использует уже имеющиеся файлы проекта и при наличии FEniCS может повторно выполнить текущий расчёт для визуализации поля решения.

## Запуск

Из корня репозитория:

```bash
streamlit run 2/presentation/app.py
```

## Зависимости

Для запуска интерфейса потребуются как минимум:

- `streamlit`
- `matplotlib`
- `numpy`
- `Pillow`

Для интерактивного построения поля решения в разделе «Визуализация решения» дополнительно нужен установленный `FEniCS`.

## Что используется из репозитория

- `2/solve.py` — постановка и решение задачи в FEniCS;
- `2/grid_template.geo` — геометрические параметры;
- `2/generate_grids.sh` — семейства вычислительных сценариев;
- `2/mesh/...` — сетки и разметка границ;
- `2/figs/5.png` — готовое изображение расчётной области;
- `2/task.py` — исходная учебная Streamlit-структура, использованная как контекст для подачи.
