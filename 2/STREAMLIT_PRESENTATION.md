# Streamlit-доклад по задаче обтекания двух цилиндров

## Запуск

Из корня репозитория:

```bash
streamlit run 2/presentation/app.py
```

## Что представляет собой приложение

Презентация оформлена как мини-доклад по численному решению задачи потенциального
обтекания двух цилиндров в плоском канале.

Структура доклада:

- титульный слайд;
- постановка задачи;
- расчётная область;
- граничные условия и параметры задачи;
- расчётные сетки;
- конечно-элементная аппроксимация;
- выбор степени полинома `p = 1, 2, 3`;
- численные решения для симметричных и несимметричных сценариев;
- сравнение результатов и циркуляции;
- краткие выводы.

## Runtime-расчёт

Поле решения и циркуляция вычисляются во время работы приложения через вызов
`solve_problem(...)` из `2/solve.py`.

Если `FEniCS` недоступен в среде запуска, приложение выводит предупреждение и не падает.

## Основные файлы

- `2/solve.py` — расчётная функция и совместимый CLI-запуск;
- `2/presentation/app.py` — Streamlit-доклад;
- `2/presentation/presentation_data.py` — чтение сеток и построение предметных визуализаций.
